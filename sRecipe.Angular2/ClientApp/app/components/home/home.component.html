Share Recipe Application Release 2

This is an team project for the course COMP231 Computer Programming Project.

According to the instructions, this project is developed by ASP.NET MVC and SQL Server features.

  • Create AngularJS 2 Application
  • Support dynamic Theme
  • Realize to Changing Theme programmably
  • Using ASP.NET Ajax, Custom Client and Server Validattion
  • Implement CRUD operation by Entity Framework for SQL Server
  • Develop productive web design page by Bootstrap
  • Connecting to Microsoft Azure SQL Server Database
  • Deploy application to Microsoft Azure
  • Controlling Source Code through Github
